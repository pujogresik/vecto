# 🧠 Vecto Core – Skill

## Overview
Vecto Core is a modular CLI agent built on Intercom architecture.  
It simulates structured execution flows with deterministic command handling.

---

## Core Capabilities

- Interactive CLI engine
- Structured command routing
- Deterministic state transitions
- Modular command handler
- Lightweight runtime (Node.js)

---

## Runtime

- Node.js (ESM compatible)
- Terminal-based execution
- No external dependencies required
